SnapSide 📸

SnapSide is a sleek, modern photography portfolio built entirely with HTML5 and CSS3. The project focuses on a "Mobile-First" design, featuring a custom Navigation Drawer and a Hamburger Menu for a clean user experience.
🚀 Features

    Navigation Drawer: A vertical slide-out menu containing links for Gallery, Exhibits, and Store.

    Pure CSS Interactivity: Hover effects and transitions built without heavy JavaScript.

    Responsive Layout: Designed using CSS Grid and Flexbox to ensure the site looks great on all devices.

    Glassmorphic UI: A translucent sidebar effect that adds a premium feel to the photography content.

🛠️ Tech Stack

    HTML5: Semantic structure for better accessibility.

    CSS3: Advanced styling using Tailwind CSS (Utility-first approach).

    Icons: FontAwesome or Heroicons for the navigation links.
