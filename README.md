# CSS Grid & Animations Learning Journey 🚀

This repository contains my progress through **CSS Part 6**, focusing on modern layout techniques with CSS Grid and web animations. As a Computer Science student, I'm documenting my transition from basic styling to complex, responsive layouts.

## 📌 Topics Covered
* **CSS Grid Foundations:** Rows, Columns, Gaps, and Template areas.
* **Grid Functions:** Mastering the `repeat()` function for clean code.
* **Advanced Layouts:** Responsive design using **Media Queries**.
* **Animations:** Keyframes, transitions, and shorthand properties.
* **Stacking Context:** Practical implementation of `z-index`.

## 🏗️ Featured Project: Pet Adoption Page
Located in the `/Pet Adoption Page` folder, this project implements a full-stack-ready frontend layout using:
- CSS Grid for the main gallery.
- Custom styling for a clean, user-friendly UI.
- Responsive design for different screen sizes.
